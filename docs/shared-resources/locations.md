# Shared Locations Documentation

## Overview

Shared locations are physical and virtual spaces accessible by both the Academic and Entertainment career systems. These spaces facilitate collaboration, resource sharing, and cross-system interactions while reducing data duplication and promoting worldbook consistency.

## Location Summary

| Name | UID | Type | Systems | Primary Use |

| University Campus | `loc_university_campus` | Educational | Both | Academic hub with performance venues |
| Grand Performance Hall | `loc_performance_hall` | Entertainment | Both | Primary performance and ceremony venue |
| Central Research Library | `loc_research_library` | Educational | Both | Academic and artistic research resources |
